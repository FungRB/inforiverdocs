Export Custom Report Themes

In Inforiver visual you can export the custom report themes as a JSON file format. A JSON file provides granular level customizations that can be easily shared and imported in a new report. Using the export custom theme option, you can standardize the reports and charts formats across all the reports.

Export Custom Theme#

  • Select the Advanced tab on the ribbon and select Export Theme.
  • Enter theme name and select Download.
  • Right Click on the the theme name and select Save Link As option.
  • Select Save and your export theme file is downloaded.

JSON Theme File Format#

The most basic JSON file has following fields:

Field/Section NameDescription
nameCustom theme file name. This is the only required field.
visualStylesExtended-format JSON file with nested formatting specifications.
chartTitleTitle field settings
annotationAnnotation configurations
headerOptionTable Header settings
inforiverPropsInforiver global visual formatting configurations
subTotalsRow and Column level sub totals setting to true or false.
exportExport settings
dataColorsHexadecimal color codes to use of data in Inforiver visuals.

Below is a templated example of the visualStyles section:

"visualStyles": {
"<visualName>": {
"<styleName>": {
"<cardName>": [
{
"<propertyName>": <propertyValue>
}
]
}
}
}

Import Custom Report theme file#

  • In the View ribbon bar, select the drop-down button in the Themes section.
  • Select Browse for themes.
  • Navigate and select your custom theme file and select Open.
  • A dialog box shows that the import was successful.
  • Select Close.

You have successfully imported the custom theme settings in the current Inforiver visual.
