The `ISNUMBER` function checks if the value refers to a number.

## Syntax

```
ISNUMBER(value)
```

## Arguments

value: A Cell or a number

## Example

To check every value in a column(Employee_id) is numeric, use the formlas as:

```javascript
ISNUMBER(Employee_id)
```

## Returns

'True' or 'False'
