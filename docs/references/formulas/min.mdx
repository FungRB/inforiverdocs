The `MIN` function returns the smallest number in a set of values. Cell (column/rows) references also accepted as arguments.

## Syntax

```javascript
MIN (value1,[value2]....)
```

## Arguments

`value1,value2 …` – Numbers or cell references of which you want the smallest value of.

## Example

Consider two rows with the sales expense and marketing expense for four quarters.

| Account           | Q1    | Q2    | Q3    | Q4    |
|-------------------|-------|-------|-------|-------|
| Sales Expense     | 10.1m | 14.5m | 13.2m | 12.4m |
| Marketing Expense | 11.3m | 12.0m | 14.6m | 13.2m |
| <b>MIN</b>        | 10.1m | 12.0m | 13.2m | 12.4m |

To calculate the minimum of the two expenses in the resultant row 'MIN', the formula should be:

```javascript
MIN(Sales Expense, Marketing Expense)
```
