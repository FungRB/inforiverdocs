The `FLOOR` function rounds a given number down to the nearest significance multiple.

## Syntax

```javascript
FLOOR (number, significance)
```

## Arguments

**number** - The number that should be rounded.
**significance** - The multiple to use when rounding.

## Example

| COLUMN1 | Formula | Result |
|---------|---------|--------|
| 10      | 4       | 8      |
| 2.5     | 1       | 2      |
| 0.156   | 0.01    | 0.15   |
| 36      | 7       | 35     |
| -5.5    | 1       | -6     |
