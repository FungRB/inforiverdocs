The `COUNTIF` function returns the count of item matches the condition.

## Syntax

```javascript
COUNTIF([list],condition)
```

## Example

To get the count of columns with value lesser than 200, use the formula as:

```javascript
COUNTIF([Sales,Finance,Inventory],"<200")
```

## Returns

The formula returns an integer.
