The `ROUND` function rounds a number to a specified number of digits.

## Syntax

```
ROUND(number,num_digits)
```

## Example

The following forumla, returns the round of 'COLUMN1' upto 2 decimal places.

```javascript
ROUND(COLUMN1, 2)
```
