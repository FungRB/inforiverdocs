The `CEILING` function returns number rounded up, away from zero, to the nearest highest multiple of significance.

## Syntax

```javascript
CEILING(number, significance)
```

## Arguments

* **number**: The number/cell reference that should be rounded.
* **significance**: The multiple to use while rounding.

## Example

To round each value in 'COLUMN1' to the highest multiple:

| COLUMN1 | Function | Result |
|---------|----------|--------|
| 10      | 4        | 12     |
| 2.5     | 1        | 3      |
| 0.156   | 0.01     | 0.16   |
| 36      | 7        | 42     |
| -5.5    | 1        | -5     |
