## Logical Functions

<table className="table">
<thead>
  <tr>
    <th>NAME</th>
    <th>SYNTAX</th>
    <th>DESCRIPTION</th>
    <th>Example</th>
    <th>Explanation</th>
  </tr>
</thead>
<tbody>
  <tr>
    <td><Link to={useBaseUrl('references/formulas/if')}>IF</Link></td>
    <td>If( logical_test, value_if_true, value_if_false )</td>
    <td>Checks whether a condition is met, and returns one value if TRUE, and another value if FALSE</td>
    <td><code>IF(PY > 0 , (AC-PY)/PY, 0 )</code></td>
    <td>Above formula will  return AC-PY% if PY is greater than 0 otherwise it returns 0</td>
  </tr>
  <tr>
    <td><Link to={useBaseUrl('references/formulas/switch')}>SWITCH</Link></td>
    <td>SWITCH ( expression, value1, result1, value2, result2, ... value_n, result_n [, default] )</td>
    <td>Evaluates; ;an expression against a list of values and returns the result corresponding to the first matching value. If there is no match, an  optional default value is returned.</td>
    <td><code>SWITCH(Region, "West", 25, "East",30, 50)</code></td>
    <td>Will return 25 if the region is West and 30 is region is East and 50 for all other regions</td>
  </tr>
  <tr>
    <td><Link to={useBaseUrl('references/formulas/and')}>AND</Link></td>
    <td>AND( logical_test1, [logical_test2], ... )</td>
    <td>Checks whether all the arguments are TRUE, and returns TRUE if all arguments are TRUE</td>
    <td><code>IF(AND(Region == "West", Category=="Urban"), 25, 30)</code></td>
    <td>Will return 25 if the region is West and category is Urban otherwise returns 30</td>
  </tr>
  <tr>
    <td><Link to={useBaseUrl('references/formulas/or')}>OR</Link></td>
    <td>OR( logical_test1, [logical_test2], ... )</td>
    <td>Checks whether any of the arguments are TRUE, and returns TRUE or FALSE. Returns FALSE only if all arguments are FALSE</td>
    <td><code>IF(OR(Region == "West", Region =="East"), 25, 30)</code></td>
    <td>Will return 25 if the region is West or East otherwise returns 30</td>
  </tr>
  <tr>
    <td><Link to={useBaseUrl('references/formulas/not')}>NOT</Link></td>
    <td>NOT( logical_test )</td>
    <td>Changes FALSE to TRUE, or TRUE to FALSE</td>
    <td><code>IF(NOT(Region == "West"), 50, 25)</code></td>
    <td>Will return 50 if the region is not West otherwise returns 25</td>
  </tr>
  <tr>
    <td><Link to={useBaseUrl('references/formulas/xor')}>XOR</Link></td>
    <td>XOR( logical_test1, [logical_test2], ... )</td>
    <td>Returns a logical 'Exclusive Or' of all arguments</td>
    <td><code>IF(XOR(Region == "West", Category=="Urban"), 25, 30)</code></td>
    <td>Will return 25 for all category in the West except for category Urban, and returns 25 for all Urban category under all region except for west, for all other items returns 30</td>
  </tr>
  <tr>
    <td><Link to={useBaseUrl('references/formulas/isnumber')}>ISNUMBER</Link></td>
    <td>ISNUMBER(value)</td>
    <td>Checks whether value is a valid number</td>
    <td><code>IF(ISNUMBER((AC-PY)/PY), (AC-PY)/PY , 0)</code></td>
    <td>Returns AC-PY/PY if the value is a number else returns 0</td>
  </tr>
  <tr>
    <td><Link to={useBaseUrl('references/formulas/ifna')}>IFNA</Link></td>
    <td>IFNA(value1,value2)</td>
    <td>Return value2 if value1 is not number else returns value1</td>
    <td><code>IFNA((AC-PY)/PY, 0)</code></td>
    <td>Returns AC-PY/PY if the value is a number else returns 0</td>
  </tr>
  <tr>
    <td><Link to={useBaseUrl('references/formulas/in')}>IN</Link></td>
    <td>IN(value,[Item1,Item2])</td>
    <td>Returns true if the given value is in list</td>
    <td><code>IF(IN(Region, ["West","East]),25, 30)</code></td>
    <td>Will return 25 if the region is West or East otherwise returns 30</td>
  </tr>
</tbody>
</table>

## Identifiers

<table>
<thead>
  <tr>
    <th>IDENTIFIER</th>
    <th>Usage</th>
    <th>DESCRIPTION</th>
    <th>Example</th>
    <th>Explanation</th>
  </tr>
</thead>
<tbody>
  <tr>
    <td>COLUMN_ID</td>
    <td>Calculated Column Formula</td>
    <td>Represents the particular column, used in column formula</td>
    <td>AC + PY</td>
    <td>Returns the sum of the column AC and PY</td>
  </tr>
  <tr>
    <td>ROW_ID</td>
    <td>Calculated Row Formula</td>
    <td>Represents the particular row, used in row formula</td>
    <td>R_WEST.R_URBAN + R_EAST.R_URBAN</td>
    <td>Returns the sum row Urban under West and East</td>
  </tr>
  <tr>
    <td>COLUMN</td>
    <td>Calculated Row Formula</td>
    <td>Represents the active column in row calculation</td>
    <td>IF(COLUMN == "AC", 5000, 6000)</td>
    <td>On using in the row formula, returns 5000 for AC column and 6000 for other columns</td>
  </tr>
  <tr>
    <td>ROW.PARENT</td>
    <td>Calculated Column Formula, Used along with identifier PARENT and </td>
    <td>Represents the active row in column calculation</td>
    <td>ROW.PARENT.AC + ROW.PARENT.PY</td>
    <td>For each row it adds the value of its parent row's AC column and PY column</td>
  </tr>
  <tr>
    <td>PARENT(Chaining)</td>
    <td>Calculated Column Formula, Used along with identifier PARENT and </td>
    <td>Multiple chaining of PARENT is possible</td>
    <td>ROW.PARENT.PARENT.AC + ROW.PARENT.PARENT.PY</td>
    <td>For each row it adds the value of its grandparent row's AC column and PY column</td>
  </tr>
</tbody>
</table>
