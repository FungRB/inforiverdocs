## Import Custom Configurations

To import and apply visual theme from an exported file:

* From the **Advanced** tab, select **Export/Import**.

* Select **Import Config** from the dropdown menu bar.

<div style={{ textAlign: "center" }}>
    <Zoom>
    <img
    alt="Upload the Config file"
    src={useBaseUrl('img/export/import-config.png')}
    />
    </Zoom>
    <div class="center"><p>Select to Upload the config file</p></div>
</div>

* Select **Upload** and navigate to the location of the exported config file.

* Select the `JSON` format exported file and select **Open**.

<div style={{ textAlign: "center" }}>
    <Zoom>
    <img
    alt="Navigate custom theme file"
    src={useBaseUrl('img/export/import-config-confirm.png')}
    />
    </Zoom>
</div>

* Select **Yes** to confirm importing the config file.

Your visual will be now updated with the data configurations from the exported file.

## Import Visual Theme

* In the **View** ribbon bar, select the drop-down button in the **Themes** section.
* Select **Browse for themes**.

<div style={{ textAlign: "center" }}>
    <Zoom>
    <img
    width="450"
    alt="Browse for Themes"
    src={useBaseUrl('img/export/browse-themes.png')}
    />
    </Zoom>
</div>

* Navigate and select your custom theme file and select **Open**.

<div class="center">
    <Zoom>
    <img
    width="450"
    alt="Navigate custom theme file"
    src={useBaseUrl('img/export/navigate-theme.png')}
    />
    </Zoom>
</div>

* A dialog box shows that the import was successful.
* Select **Close**.

<div style={{ textAlign: "center" }}>
    <Zoom>
    <img
    width="450"
    alt="Custom theme imported"
    src={useBaseUrl('img/export/theme-imported.png')}
    />
    </Zoom>
</div>

You have successfully imported the custom theme settings in the current Inforiver visual.
