In Inforiver visual you can export the custom report themes as a `JSON` file format.
A `JSON` file provides granular level customizations that can be easily shared and imported in a new report.
Using the export custom theme option, you can standardize the reports and charts formats across all the reports.

## Export Custom Theme

* Select the **Advanced** tab on the ribbon and select **Export Theme**.

    <Zoom>

    </Zoom>
</div>

* **Enter theme name** and select **Download**.

        src={useBaseUrl('img/export/theme-name.png')}

* Select **Save** and your export theme file is downloaded.

## JSON Theme File Format

The most basic `JSON` file has following fields:

| Field/Section Name | Description                                                      |
|--------------------|------------------------------------------------------------------|
| `name`             | Custom theme file name. This is the only required field.         |
| `visualStyles`     | Extended-format JSON file with nested formatting specifications. |
| `chartTitle`       | Title field settings                                             |
| `annotation`       | Annotation configurations                                        |
| `headerOption`     | Table Header settings                                            |
| `inforiverProps`   | Inforiver global visual formatting configurations                |
| `subTotals`        | Row and Column level sub totals setting to true or false.        |
| `export`           | Export settings                                                  |
| `dataColors`       | Hexadecimal color codes to use of data in Inforiver visuals.     |

Below is a templated example of the visualStyles section:

```json "visualStyles template"
"visualStyles": {
    "<visualName>": {
        "<styleName>": {
            "<cardName>": [
                {
                    "<propertyName>": <propertyValue>
                }
            ]
        }
    }
}
```

## Import Custom Report theme file

* In the **View** ribbon bar, select the drop-down button in the **Themes** section.
* Select **Browse for themes**.

<div style={{ textAlign: "center" }}>
    <Zoom>
    <img
    width="450"
    alt="Browse for Themes"
    src={useBaseUrl('img/export/browse-themes.png')}
    />
    </Zoom>
</div>

* Navigate and select your custom theme file and select **Open**.

<div class="center">
    <Zoom>
    <img
    width="450"
    alt="Navigate custom theme file"
    src={useBaseUrl('img/export/navigate-theme.png')}
    />
    </Zoom>
</div>

* A dialog box shows that the import was successful.
* Select **Close**.

<div style={{ textAlign: "center" }}>
    <Zoom>
    <img
    width="450"
    alt="Custom theme imported"
    src={useBaseUrl('img/export/theme-imported.png')}
    />
    </Zoom>
</div>

You have successfully imported the custom theme settings in the current Inforiver visual.
