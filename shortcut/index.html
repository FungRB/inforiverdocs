Inforiver Visual Shortcuts

Shortcut KeyFunctional Description
Navigation← ↑ → ↓Navigates in the grid
  • `← →` wont work if you are in row selection
  • `↑ ↓` wont work if you are in column selection
`Ctrl` + `← ↑ → ↓`Goes to the edge of the respective direction inside the UI(within the pagination interval)
`Shift`+ `← ↑ → ↓`Multi select continuously (row,column)
`Ctrl` + select(row,column)Multi select
`Shift` + select(row,column)Selects all the intermediate rows/columns from source to target like excel
Formatting`Ctrl` + `b`Bolds the selection(row,column,cell)
`Ctrl` + `i`Italicize the selection(row,column,cell)
`Ctrl` + `l`Bottom line style in selection(row,column,cell)
`Ctrl` + `Shift`+`%` converts the selection(row,column,cell) into percentage
Tabs`alt` + `h`Home Tab
`alt` + `a` Advanced Tab
`alt` + `e`Export Tab
Number Format`alt` + `n`Number Formatting Dropdown - Not valid on runtime
`alt` + `n+1`Default
`alt` + `n+2`Unified
`alt` + `n+3`Native
`alt` + `n+4`Auto
Scaling`alt` + `s`Scaling Dropdown
`alt` + `s+1`Thousand - only on selection
`alt` + `s+2`Million - only on selection
`alt` + `s+3`Billion - only on selection
`alt` + `s+4`Trillion - only on selection
`alt` + `s+5`None - only on selection
Modals`alt` + `d`Total Popup
`alt` + `j`Alt J - Prefix/ Suffix Popup - only on selection 
`alt` + `n` Top N popup only with any col selection
`alt` + `p`PDF popup
`alt` + `x+l`XLS popup
Conditional Format`alt` + `f`Conditional Formatting Side panel Toggle
`alt` + `f+1`Static Segmentation Toggle
`alt` + `f+2`Heat map Toggle
`alt` + `f+3`ABC Classification Toggle
Aggregation`alt` + `g`Aggregation Type Dropdown
`alt` + `g+1`Native
`alt` + `g+2`Sum
`alt` + `g+3`Min
`alt` + `g+4`Max
`alt` + `g+5`Avg
`alt` + `g+6`Std Dev
Pagination`alt` + `p+0`Pagination dropdown
`alt` + `p+1`Responsive
`alt` + `p+2`Fixed Rows
`alt` + `p+3`None
Templates`alt` + `t` Template side panel toggle
`alt` + `t+1`T01 Template
`alt` + `t+2`T02 Template
`alt` + `t+3`T03 Template
`alt` + `t+4`T04 Template
`alt` + `t+5`T05 Template
`alt` + `t+6`Matrix Template
`alt` + `t+7`Spreadsheet Template
`alt` + `t+8`Simple Template
`alt` + `t+9`Bars Template
`alt` + `t+10`Waterfall Template
Menu and Side Panels`alt` + `b`Navigation Dropdown - Not valid on runtime
`alt` + `o`Sign Type Dropdown - Not valid on runtime
`alt` + `r` Reorder Toggle
`alt` + `i`Invert row dropdown - Not valid on runtime
`alt` + `z`Insert  col dropdown  - Not valid on runtime
`alt` + `m`Row Multi select Toggle
`alt` + `k`Breaks Dropdown - Not valid on runtime
`alt` + `q`Alert Side panel toggle - Not valid on runtime
`alt` + `u`Audit Side panel toggle - Not valid on runtime
`alt` + `right`Expand selected row level
`alt` + `left`Collapse selected row level
`ctrl` + 'shift' + `right`Expand all
`ctrl` + 'shift' + `left`Collapse all
`alt` + `l+1...to...9`Row Selection
`alt` + `c+1...to...9`Column Selection
