IPMT

The IPMT function returns the interest payment for a given period for an investment, based on periodic, constant payments and a constant interest rate.

Syntax#

IPMT (Rate, Per, Nper, PV, [FV], [Type])

Rate â€“ The interest rate per period.

Per â€“ The period for which you want to find the interest and must be in the range 1 to Nper.

Nper â€“ The total number of periods.

PV â€“ The present value or initial investment. Cash outflows are considered as negative and cash inflows as positive.

FV â€“ The future or residual value. This is an optional argument and if omitted, it is considered to be zero.

Type â€“ Indicates when the payments are made. Type is zero if payments are made at the end of the period and non-zero if payments are made at the start of the period. This is an optional argument and when omitted, it is considered to be zero.

Example#

Consider a $10,000 loan at an annual rate of 10% that is to be paid off in 3 years. All payments are made at the beginning of the month.

To calculate the interest payment for a specified period, say 8, the formula should be

IPMT (Rate/12, Period, Number of Years*12, Loan Amount, 0, 1).

IPMT is calculated to be $68.45.
