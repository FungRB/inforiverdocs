NODE PROPERTIES

[NODE].PARENT#

[NODE].PARENT refers to the parent node of the NODE.

Example#

If the node Sales has child nodes, East, West, South and Central, the node Sales can be referred to as East.PARENT, West.PARENT, South.PARENT or Central.PARENT.

[NODE].CHILDREN#

[NODE].CHILDREN refers to the children nodes of the NODE.

Example#

If the node Sales has child nodes, East, West, South and Central, the child nodes of Sales can be referred to using Sales.CHILDREN.

[NODE].VISIBLE_CHILDREN#

[NODE].VISIBLE_CHILDREN refers to the children nodes of the NODE that are not hidden.

[NODE].HIDDEN_CHILDREN#

[NODE].HIDDEN_CHILDREN refers to the children nodes of the NODE that are hidden.

[NODE].ID#

[NODE].ID returns the node ID.

[NODE].ACT#

[NODE].ACT refers to the Simulated Series of the node.

Example#

If a node N2 is defined as N1.ACT, the node N2 will be assigned the simulated series value of N1.

[NODE].TGT#

[NODE].TGT refers to the Comparison Series of the node.

Example#

If a node N2 is defined as N1.TGT, the node N2 will be assigned the comparison series value of N1.

[NODE].BASE#

[NODE].BASE refers to the Baseline Series of the node.

Example#

If a node N2 is defined as N1.BASE, the node N2 will be assigned the baseline series value of N1.

Consider a node with the following data.

Period Node N1Baseline SeriesComparison SeriesSimulated Series
Week1201525
Week2201525
Week3201525
Week4201525

The results of the calculations are shown below.

Period Node N2N1.ACTN1.TGTN1.BASE
Week1251520
Week2251520
Week3251520
Week4251520

[NODE].ALL_PERIODS#

[NODE].ALL_PERIODS refers to all period values.

[NODE].SIM_PERIODS#

[NODE].SIM_PERIODS refers to simulated period values.

Example#

The periods in a model are defined from Jan to Dec and the active simulation period is from Jun to Dec. Node N1 has the following values:

PeriodJanFebMarAprMayJunJulAugSepOctNovDec
Values101520101525301520253530

[NODE].ALL_PERIODS returns [10,15,20,10,15,25,30,15,20,25,35,30].

[NODE].SIM_PERIODS returns [25,30,15,20,25,35,30].

[NODE].ACT.ALL_PERIODS#

[NODE].act.ALL_PERIODS returns all period values of the simulated series.

[NODE].TGT.ALL_PERIODS#

[NODE].tgt.ALL_PERIODS returns all period values of the comparison series.

[NODE].BASE.ALL_PERIODS#

[NODE].base.ALL_PERIODS returns all period values of the baseline series.

Example#

The periods in a model are defined from Jan to Dec and the active simulation period is from Jun to Dec. Node N1 has the following values:

PeriodJanFebMarAprMayJunJulAugSepOctNovDec
Baseline101520101525301520253530
Comparison151515151515151515151515
Simulated101520101530361824304236

[NODE].ACT.ALL_PERIODS returns [10,15,20,10,15,30,36,18,24,30,42,36].

[NODE].TGT.ALL_PERIODS returns [15,15,15,15,15,15,15,15,15,15,15,15].

[NODE].BASE.ALL_PERIODS returns [10,15,20,10,15,25,30,15,20,25,35,30].

[NODE].TOTAL#

[NODE].TOTAL returns the all period aggregated value of the node.

[NODE].PTD#

[NODE].PTD returns the period till date aggregated value of the node.

[NODE].SEL#

[NODE].SEL returns the simulation period aggregated value of the node.

Example#

The periods in a model are defined from Jan to Dec and the active simulation period is from Jun to Dec. Node N1 has the following values:

PeriodJanFebMarAprMayJunJulAugSepOctNovDec
Values101520101525301520253530

[NODE].TOTAL returns 250.

[NODE].PTD returns 70.

[NODE].SEL returns 180.

[NODE].RANGE(START_INDEX,END_INDEX)#

[NODE].RANGE(start_index,end_index) returns selected range values of the node.

[NODE].ACT.RANGE(START_INDEX,END_INDEX)#

[NODE].ACT.RANGE(start_index,end_index) returns selected range values of Simulated Series.

[NODE].BASE.RANGE(START_INDEX,END_INDEX)#

[NODE].BASE.RANGE(start_index,end_index) returns selected range values of Baseline Series.

[NODE].TGT.RANGE(START_INDEX,END_INDEX)#

[NODE].TGT.RANGE(start_index,end_index) returns selected range values of Comparison Series.

Example#

The periods in a model are defined from Jan to Dec. Node N1 has the following values:

PeriodJanFebMarAprMayJunJulAugSepOctNovDec
Baseline101520101525301520253530
Comparison151515151515151515151515
Simulated101520101530361824304236

[NODE].RANGE(2,4) returns a node with baseline as [15,20,10] and comparison as [15,15,15].

[NODE].ACT.RANGE(9,12) returns a node with baseline and comparison as [24,30,42,26].

[NODE].TGT.RANGE(2,4) returns a node with baseline and comparison as [15,15,15].

[NODE].BASE.RANGE(9,12) returns [20,25,35,30].

[NODE].GET(PERIOD_INDEX)#

[NODE].GET(period_index) returns the selected period value of the node.

[NODE].BASE.GET(PERIOD_INDEX)#

[NODE].BASE.GET(period_index) returns the selected period value of the Baseline Series.

[NODE].TGT.GET(PERIOD_INDEX)#

[NODE].TGT.GET(period_index) returns the selected period value of the Comparison Series.

Example#

The periods in a model are defined from Jan to Dec. Node N1 has the following values:

PeriodJanFebMarAprMayJunJulAugSepOctNovDec
Baseline101520101525301520253530
Comparison151515151515151515151515
Simulated101520101530361824304236

[NODE].TGT.GET(9) returns the value 15.
